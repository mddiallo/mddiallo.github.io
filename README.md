# mddiallo.github.io

Personal developer portfolio built with [Jekyll](https://jekyllrb.com/) and
served by [GitHub Pages](https://pages.github.com/) at
<https://mddiallo.github.io>.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://127.0.0.1:4000>.

## Editing content

- **Site metadata, social links** → `_config.yml`
- **Skills** → `_data/skills.yml`
- **Projects** → `_data/projects.yml`
- **Avatar / CV** → drop files into `assets/img/avatar.jpg` and `assets/cv.pdf`

## Deploying

Push to `main` — GitHub Pages rebuilds automatically.
